# Backend Punto de Venta

API REST para un sistema de punto de venta de marquesitas, desarrollada con Node.js, Express y MySQL.

---

## Tecnologías y Dependencias

Este proyecto utiliza las siguientes librerías para el desarrollo del backend:

### Dependencias principales

- **express** → Framework para la creación del servidor y API REST  
- **morgan** → Middleware para el registro de peticiones HTTP (logs)  
- **cors** → Permite la comunicación entre el frontend y el backend  
- **sequelize** → ORM para la gestión de la base de datos MySQL  
- **mysql2** → Driver para la conexión con MySQL  
- **jsonwebtoken** → Implementación de autenticación mediante JWT  
- **dotenv** → Manejo de variables de entorno  
- **bcryptjs** → Encriptación de contraseñas  
- **express-validator** → Validación de datos en las peticiones  

---

### Dependencias de desarrollo

- **nodemon** → Reinicio automático del servidor durante el desarrollo
